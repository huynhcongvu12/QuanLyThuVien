# BẢN GIAO DIỆN LUXURY / XỊN 2026

Bản này đã được nâng cấp giao diện theo phong cách video ngắn hiện đại: glassmorphism, neon gradient, dashboard cinematic, card nổi, hiệu ứng hover 3D, spotlight theo chuột, nút cuộn nổi và animation khi kéo trang.

## File thêm mới
- web/assets/luxury-ui.css
- web/assets/luxury-ui.js
- web/assets/image/lux-wave.svg
- web/assets/image/lux-book-cover-1.svg đến lux-book-cover-5.svg
- web/components/luxury-header.jsp
- web/LuxuryPreview.jsp

## File đã nhúng giao diện mới
- web/Login.jsp
- web/Register.jsp
- web/Customer.jsp
- web/Staff.jsp
- web/Admin.jsp

## Cách chạy
1. Mở project bằng NetBeans.
2. Clean and Build.
3. Run project.
4. Nếu trình duyệt còn giao diện cũ, nhấn Ctrl + F5.
5. Có thể mở thêm LuxuryPreview.jsp để xem nhanh style mới.

## Lưu ý
Giao diện mới chủ yếu cập nhật UI/CSS/JS, không phá luồng Servlet/JSP cũ. Các chức năng mượn/trả/phạt vẫn giữ nguyên theo bản trước.
