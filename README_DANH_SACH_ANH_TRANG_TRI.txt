LƯU Ý CHẠY PROJECT
1. Mở project bằng NetBeans.
2. Clean and Build.
3. Run lại trên Tomcat.
4. Nếu chưa thấy ảnh, bấm Ctrl + F5 để reload trình duyệt không dùng cache.
